# ESPExplorerAE

ESPExplorerAE is an F4SE plugin for Fallout 4 Next-Gen that adds an in-game explorer for plugins, forms, player actions, and diagnostics.

It is currently built for Fallout 4 runtime v1.11.191.0.

The in-game UI lets you browse records, inspect details, search across plugins, run quick player actions, and review logs without leaving the game.

## Translation Note

Shipped translations were produced with LLM assistance and may not be completely accurate yet.

If you notice incorrect wording, missing context, or awkward phrasing in any language file, treat English as the reference and feel free to report or contribute fixes.

## Scope

- Fallout 4 Next-Gen support is the current priority.
- Localization, controller support, and UI stability are still being refined.
- Packaging includes the plugin DLL plus the bundled language and font assets.

## FAQ

### Will this be ported to Skyrim or Starfield?

Maybe, but not before the Fallout 4 version is stable.

### Will this be backported to older Fallout 4 versions?

Yes, that is planned, but not right now. The current focus is stabilizing the Next-Gen build first.

### Are the translations final?

No. They are usable, but they still need review and cleanup.

### Does this replace xEdit, CK, or other external tools?

No. It is meant to be a fast in-game inspection and action tool, not a full authoring pipeline.

### Does it support controller use?

Yes, there is controller navigation support, but keyboard and mouse are still the primary baseline for troubleshooting and validation.

### Can this break saves?

The plugin is designed for runtime inspection and utility actions, but any mod that changes live game state should be used carefully and on saves you are comfortable testing with.
